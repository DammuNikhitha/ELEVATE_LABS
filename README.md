# ELEVATE_LABS – AI & Machine Learning Internship Tasks

This repository contains all the tasks completed as part of my AI & Machine Learning Internship at Elevate Labs.
Each task focuses on different concepts of data analysis, machine learning, and model building using Python and Scikit-learn.

The repository is organized in a structured and sequential manner for easy understanding and navigation.

---

1. Understanding Dataset & Data Types
2. Data Cleaning & Missing Value Handling
3. Exploratory Data Analysis – Iris Dataset
4. Feature Encoding & Scaling
5. Heart Disease Prediction
6. Linear Regression – House Price Predictio
7. Titanic Survival Prediction using Logistic Regression
8. Decision Tree – Bank Marketing Subscription Prediction
9. Credit Card Fraud Detection using Random Forest
10. KNN – Handwritten Digit Classification
11. SVM – Breast Cancer Classification
12. KMeans Customer Segmentation
13. PCA – Dimensionality Reduction
14. Model Comparison & Best Model Selection
15. End-to-End Machine Learning Pipeline
