# AI-ML-Internship-Task-7
## Titanic Survival Prediction using Logistic Regression

## Description
This project predicts passenger survival on the Titanic dataset using Logistic Regression.

## Dataset
- Source: Kaggle Titanic Dataset
- Target Variable: Survived (0 = No, 1 = Yes)

## Steps
- Data cleaning
- Encoding categorical features
- Feature scaling
- Model training
- Evaluation using ROC & Confusion Matrix

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Final Outcome

- Successfully trained Logistic Regression model
- Evaluated using Accuracy, Precision, Recall, F1-score
- Visualized Confusion Matrix
- Plotted ROC Curve and calculated AUC score

This task helped in understanding binary classification and model evaluation techniques.
