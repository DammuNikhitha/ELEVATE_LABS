# AI-ML-Internship-Task-11
## Task 11: SVM – Breast Cancer Classification

This project uses Support Vector Machine (SVM) to classify breast cancer tumors as malignant or benign.

### Tools Used:
- Python
- Scikit-learn
- Matplotlib

### Dataset:
- Sklearn Breast Cancer Dataset (load_breast_cancer)

### Steps Performed:
1. Loaded breast cancer dataset from sklearn.
2. Applied StandardScaler for feature normalization.
3. Split data into training and testing sets.
4. Trained SVM with linear and RBF kernels.
5. Tuned hyperparameters using GridSearchCV.
6. Evaluated model using confusion matrix and classification report.
7. Plotted ROC curve and calculated AUC score.
8. Saved trained model pipeline for reuse.
